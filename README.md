# Udemy Clone - MERN Course Platform

**PilotGroup5** - Full Stack Course Management System

---

## Tech Stack

### Frontend (`/frontend`)
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS (TBD: Tailwind/Material-UI)
- **State Management**: TBD (Context API/Redux)

### Backend (`/backend`)
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB + Mongoose
- **Validation**: Express Validator
---

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend (When Ready)
```bash
cd backend
npm init -y
npm install
cp .env.example .env
npm run dev
```
